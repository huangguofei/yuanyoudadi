关于源优大地

走遍大地，优选好茶。

广东源产优品农业科技有限公司创立于2017年1月，是互联网时代背景下，集线下体验，线上消费于一体的O2O茶商。源优大地商品开发团队走遍中国大地，在每年最适宜采茶的几十天内，从国家地理标志保护产区精选上等芽叶，经大师级制茶人精心呵制，再送往全球公认的瑞士SGS检测中心，确保每罐茶拥有世人赞许的品质。

长久以来，好茶没有走近大众，大多数国人没享受到好茶所赋予的健康、品味生活。因此，源优大地要做的事，就是回归！
让好茶回归生活；让价格回归透明；让生态回归自然；让选茶回归简单；让品茶回归安心
