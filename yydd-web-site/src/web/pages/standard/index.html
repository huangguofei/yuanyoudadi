色、香、味、型

通过科学仪器，检测

感官判断

感官判断需要一定的专业知识，非大众在短时间内所能掌握，这也是茶叶价格不透明的重要原因。
源优大地的评测团专家来自 中国茶叶研究所及广东省茶叶研究所，
为您严格把控每一款茶的品质。

科学检测

借助科学仪器及检测方法，查验茶叶成分，判定茶叶中是否含有对人体不利物质。这些物质与种植环境、种植方法、加工环境及仓储条件等有关。
